This Privacy Policy describes how the App handles user data. ### 1. Data Collection The App does not actively collect, transmit, or store any personal user data. All features operate locally on the user's device. ### 2. Local Storage The App stores the following data locally on the user's device to provide normal functionality: - Routes imported or created by the user (GPX routes, coordinate lists) - Application settings (floating window scale, speed preferences, API keys) - OpenRouteService API Key (if the user enables realistic route generation) This data is stored only on the user's device and is not uploaded to any server. All local data is removed when the user uninstalls the App. ### 3. Third-Party Services The App communicates with third-party services only in the following cases: 1. **Google Maps SDK**: Used for map display. When the user opens the map interface, Google Maps SDK communicates with Google servers to retrieve map tiles. This is governed by Google's privacy policy (https://policies.google.com/privacy). 2. **OpenRouteService**: Only when the user actively uses the "Realistic Route Generation" feature, the App sends coordinates and distance parameters to OpenRouteService servers to retrieve real-road routes. This feature requires users to obtain their own API key at openrouteservice.org. Their privacy policy: https://openrouteservice.org/privacy/. ### 4. Location Permissions The App may request location permissions in system settings to properly function as an Android mock location provider. The App does not collect, transmit, or store the device's real location. ### 5. Children's Privacy The App is not designed for children under 13, and does not knowingly collect personal information from children. ### 6.
